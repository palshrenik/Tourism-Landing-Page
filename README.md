# ✈️ Travel - Tourism Landing Page

A modern and responsive tourism landing page designed to inspire travelers to explore amazing destinations. Built using pure HTML and CSS with a clean interface, immersive background video, destination showcase cards, and a contact section.

## 🌟 Features

- 🎥 Full-screen hero section with background video
- 🧭 Fixed navigation bar with smooth scrolling
- 🏝️ Popular destination cards
- 📖 About section introducing the platform
- 📞 Contact information and contact form
- 📱 Responsive layout for desktop and mobile devices
- 🎨 Clean and modern UI design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Design
- Flexbox
- CSS Grid

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Tourism-Landing-Page.git
```

2. Open the index folder.

3. Launch `index.html` in your preferred web browser.

No additional dependencies or installations are required.
